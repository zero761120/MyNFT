# My NFT Android app
This project uses [AAC](https://github.com/android/architecture-components-samples) and 3rd parties to present NFT collectibles

# AAC(Android Archicture Components)
- [Lifecycle-aware components](https://developer.android.com/topic/libraries/architecture/lifecycle)
- [ViewModels](https://developer.android.com/topic/libraries/architecture/viewmodel)
- [LiveData](https://developer.android.com/topic/libraries/architecture/livedata)
- [Coroutines](https://github.com/Kotlin/kotlinx.coroutines)
- [Paging](https://github.com/googlecodelabs/android-paging)
- [DataBinding](https://github.com/googlecodelabs/android-databinding)

# Third parties
- [Retrofit](https://github.com/square/retrofit)
- [Glide](https://github.com/bumptech/glide)
- [Web3j](https://github.com/web3j/web3j)

# OpenSea API
- [OpenSea Doc](https://docs.opensea.io/reference/api-overview)

# EtherScan API
- [EtherScan Doc](https://docs.etherscan.io/api-endpoints/accounts)
